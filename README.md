# Honeypot & SIEM Lab

This repository documents my hands-on honeypot and SIEM lab work using **T-Pot, Elastic, Kibana, and Suricata**. The goal of this lab is to collect attacker telemetry, review security events, analyze suspicious traffic, and document findings in a SOC-style format.

## Purpose

This lab is designed to help me practice real-world blue-team skills, including:

- Reviewing security alerts
- Analyzing attacker behavior
- Investigating suspicious IP addresses
- Understanding network-based detections
- Documenting evidence and findings
- Recommending containment and remediation steps

## Technology Stack

- **T-Pot** for honeypot deployment
- **Elastic** for log collection and search
- **Kibana** for dashboards and visualization
- **Suricata** for network detection and alerting
- **Threat intelligence lookups** for IP and indicator review

## What This Repository Will Include

This repository will include selected analysis writeups such as:

- Suricata alert investigations
- Brute-force activity analysis
- Port scanning and reconnaissance review
- Suspicious IP investigation
- Kibana dashboard observations
- Attacker behavior summaries
- MITRE ATT&CK mapping when applicable
- Recommended response actions

## Investigation Format

Each analysis will follow a consistent structure:

1. **Scenario Summary**
2. **Tools Used**
3. **Key Evidence**
4. **Alert or Log Analysis**
5. **Findings**
6. **MITRE ATT&CK Mapping**
7. **Recommended Remediation**
8. **Lessons Learned**
9. **Interview Talking Point**
